# Resource Recommendation Agent - Frontend

智能学习资源推荐系统前端。

## 技术栈

- Vue 3
- Element Plus
- Vite
- Axios

## 安装

```bash
cd frontend
npm install
```

## 运行

```bash
npm run dev
```

访问 http://localhost:3000

## 构建

```bash
npm run build
```

## 功能

- 输入学习需求
- 实时查看推荐进度
- 查看学习主题
- 查看推荐资源
- 查看学习路径
- 查看练习任务
- 标记资源已学习
- 收藏资源
